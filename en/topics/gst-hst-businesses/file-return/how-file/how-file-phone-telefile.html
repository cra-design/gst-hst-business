By phone (TELEFILE) – File your GST/HST return

Qualifying GST/HST registrants can use GST/HST TELEFILE to file their GST/HST returns directly with the Canada Revenue Agency (CRA) over the telephone.

On this page

  • What you can and cannot do
  • Before you start
  • How to file

What you can and cannot do

Most GST/HST registrants across Canada and the United States can file a regular return using this method.

You can:

  • Meet electronic filing requirements
  • Get a confirmation number right away
  • File without an online CRA account

You cannot:

  • File without a GST/HST access code
  • File elections
  • File a return for an account administered by Revenu Québec
  • File a combined Form RC7200, GST/HST and QST return for selected listed financial institutions
  • File a return for a non-resident
  • File a simplified GST/HST return for a non-resident digital-economy business
  • Include schedules
  • File Form GST106, Information on Claims Paid or Credited for Foreign Conventions
  • File rebates (refer to Claim a GST/HST rebate)

File electronically and send forms by mail

If you want to use TELEFILE for your return but need to file an additional form or rebate claim, you can include the amount on your electronic return and mail the form to the applicable tax centre.

Make sure the tax centre receives the form on or before the due date of your return.

If you cannot use TELEFILE

Choose another filing method that works for you.

Before you start

If you plan to file your return by phone using TELEFILE, make sure you:

  • Know what you can and cannot file using this method
  • Have your four-digit access code
  • Have your Business Number (BN), reporting period "from" and "to" dates, and the rest of the information you need to complete your return
  • Are aware of the hours of service for electronic GST/HST filing

How to file

  1. Dial the toll-free telephone number 1-800-959-2038
    • An automated system will guide you through the process
    • The system will ask you to enter your tax information using your telephone keypad
  2. Use the following keys on your phone to make selections during the filing process:
    • Press "*" to have a message repeated
    • Press "1" to answer yes or "0" to answer no
    • Press "#" when you have finished entering an amount
  3. If you hang up at any time before confirming that you want to file your return, the information you entered will not be saved
    • You can hang up and call again up to a maximum of 6 times
  4. Enter all the digits of your Business Number, but do not enter the letters "RT"
  5. The system repeats each entry to help you file an accurate return
    • Make any necessary corrections before you confirm the information you have entered
  6. Continue entering and confirming each required amount for your return
  7. Once all of the information has been entered, you will be asked to confirm that the information you entered is correct
    • If you answer yes, we will consider your return received for processing
    • If you answer no, you can review and change any of the amounts you have entered
    • You can also cancel filing your return without saving your information by hanging up before you confirm (maximum of 6 calls)
  8. If your return is received, the automated system will immediately issue you a six-digit confirmation number
    • Write down your confirmation number and keep it with your records
  9. If you have an amount owing or are expecting a refund:
    • If you have an amount owing, find out how to make a payment to the CRA
    • If you are expecting a refund, speed up your deposit by setting up direct deposit
